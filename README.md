## Expense Tracker App

This is an open-source Expense Tracker Android application that allows users to manage their daily expenses efficiently.

## Features
Add new expenses
Update existing expenses
Delete expense
Search expenses easily
View total expenses
Display maximum expense (highest spending)
Display minimum expense (lowest spending)
Show total expense based on search results
## Functionality
Users can enter expense details such as title, amount, category, and date. All data is stored locally, and the app updates the list in real time. The search feature allows users to filter expenses and calculate the total for the filtered results.
